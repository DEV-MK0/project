**Projekt für die Taupuntklüftung**

[Screencast_20260408_203058.webm](https://github.com/user-attachments/assets/93c3b926-8d5d-4763-a669-577c8fac1643)

**Mindestanforderungen:**
- 2 Temperatur/Luftfeuchtigkeit Sensor
- Werte Messen
- Ventilator über Ergebnis von Taupunktberechnung an-/ausschalten
- Ergebnisse als .csv speicherbar.
- Ergebnisse in einer Datenbank speichern.
- Anzeige über eine Website.

Installation:
- python -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
- deactivate

**GitHub CLI:**
- git add .
- git commit -m "commit message"
- git push
- git pull
- Revert: git reset --hard HEAD
